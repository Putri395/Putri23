Tentang Website Ini

Website ini dibuat sebagai media personal untuk memperkenalkan diri saya secara lebih luas. Anda dapat menemukan berbagai informasi mengenai profil saya, foto-foto kegiatan, berita terbaru, serta informasi lainnya yang saya bagikan.

Website ini dirancang menggunakan HTML, CSS, dan Bootstrap agar tampilannya modern, sederhana, dan mudah diakses di berbagai perangkat.

Silakan jelajahi halaman-halaman yang tersedia untuk mengenal saya lebih dekat.
